# Quizzler Flutter App

A simple Flutter quiz application that asks true/false questions and shows the user’s score using check and close icons.

## Features

* True / False quiz questions
* Score tracking with icons
* Simple and clean dark UI
* Built using Flutter and Dart
* Uses a separate `quiz_brain.dart` file to manage quiz logic

## Screens

The app displays one question at a time.
The user can choose either **True** or **False**.
After each answer, the score row updates with a green check or red close icon.

## Technologies Used

* Flutter
* Dart
* Material Design

## How to Run

1. Clone the repository:

```bash
git clone your-repository-link
```

2. Go to the project folder:

```bash
cd quizzler
```

3. Get the Flutter packages:

```bash
flutter pub get
```

4. Run the app:

```bash
flutter run
```
